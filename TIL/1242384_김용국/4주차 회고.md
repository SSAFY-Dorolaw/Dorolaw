      -   join 할때만 처리하기

  -   Repository에 @Query(distinct + fetch join)

  -   getReferenceById를 사용해 프록시 객체를 활용

      -   Answer Entity에 변수는 아래와 같이 Request로 되어있음

          ```

          @JoinColumn(name = "request_id", nullable = false)
          private Request request;
          ```
